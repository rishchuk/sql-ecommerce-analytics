# E-commerce SQL Analytics Database

## About the project
This project demonstrates my ability to design and implement a relational database for an e-commerce system using MySQL 8.0+.

The goal of the project is to create a realistic database structure that supports:

- customers management
- product catalog
- categories
- order processing
- order items
- payments
- analytical queries
